## Installation

Clone the repository:
```bash
git clone https://github.com/FannyGautierr/rendu-git-en-entreprise.git
```

Navigate to the project folder:
```bash
cd rendu-git-en-entreprise
```

Install the dependencies:
```bash
npm install
```

Run the project:
```bash
npm dev
```

## Tests

Run the tests:
```bash
npm test
```

## Lint

Run the linter:
```bash
npm run lint
```

## Technologies

- Vite
- TypeScript
- TailwindCSS
